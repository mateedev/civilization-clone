# PokeAPI

## Entidades

### Pokemon

Los Pokemones son bichitos de Nintendo® bla bla...
Tienen las siguientes características:

- Nombre
- Evolución
- Tipo
- Rareza

#### Endpoints

- GET /api/v1/pokemons  # Obtener todos los pokemons
- GET /api/v1/pokemons/:id  # Obtener un pokemon
- POST /api/v1/pokemons  # Crear un pokemon
- PUT /api/v1/pokemons/:id  # Actualizar un pokemon
- DELETE /api/v1/pokemons/:id  # Borrar un pokemon
